# Spec: Sample Feature — Task List

**Status:** Approved  
**Created:** 2026-07-05

## Summary

Allow users to view, add, and mark tasks as complete.

## Acceptance Criteria

- [ ] User sees an empty state when no tasks exist
- [ ] User can add a task with title (max 200 chars)
- [ ] User can mark a task complete
- [ ] Completed tasks show visual distinction

## Out of Scope

- Due dates, priorities, user accounts

## Architecture Notes

ADR not required — single-page UI with local state for demo.
