# Spec: Sample Feature — Quick Notes

**Status:** Approved  
**Created:** 2026-07-05

## Summary

Allow users to write and save a plain-text note locally.

## Acceptance Criteria

- [ ] User sees a text area and save button
- [ ] Saved note persists after page reload (localStorage)
- [ ] Empty note shows validation message

## Out of Scope

- Rich text, sync, auth

## Architecture Notes

No ADR required for demo scope.
